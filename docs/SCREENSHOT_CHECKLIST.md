# Screenshot Checklist

Create screenshots under `docs/screenshots/` before publishing the repository.

Required images:
- Dashboard overview.
- Login modal.
- Incidents workflow.
- Incident timeline.
- AI incident summary interaction.
- Systems page.
- Logs page.
- Responses page.
- Users page.
- Vulnerabilities page.
- Dashboard charts.
- Swagger/OpenAPI docs.
- Architecture diagram.
- ER diagram.
- Mobile dashboard view.
- Dark theme detail shot.

Recommended naming:
```text
dashboard.png
login.png
incidents.png
incident-timeline.png
ai-summary.png
systems.png
logs.png
responses.png
users.png
vulnerabilities.png
charts.png
swagger.png
architecture.png
er-diagram.png
mobile.png
dark-theme.png
```

Browser verification checklist:
- No console errors on page load.
- Login succeeds.
- Logout returns to login overlay.
- Tables load after authentication.
- Create/update/delete buttons work.
- Search, filters, sorting, pagination, and exports work.
- Charts render on dashboard.
- Mobile viewport does not overlap text or controls.
