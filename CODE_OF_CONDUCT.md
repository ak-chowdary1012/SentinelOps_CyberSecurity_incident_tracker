# Code of Conduct

## Our Standard
This project is maintained as a professional portfolio repository. Contributions and discussions should be respectful, technically grounded, and focused on improving the software.

## Unacceptable Behavior
- Harassment, discrimination, or personal attacks.
- Publishing credentials, private data, or exploit details irresponsibly.
- Disruptive comments unrelated to the project.

## Enforcement
Maintainers may edit, reject, or remove contributions that violate this standard.
